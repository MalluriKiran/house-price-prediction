# House Price Prediction Web App

This is a **Flask-based web application** that predicts house prices using a **machine learning model**. The app takes input features from the user and outputs the predicted price of a house.  

---

## Features

- User-friendly web interface with HTML forms
- Predicts house prices using a pre-trained ML model (`model.pkl`)
- Easy deployment using **Waitress** for Windows or cloud platforms like **Render.com**
- No dataset file required for prediction; only the trained model is used
